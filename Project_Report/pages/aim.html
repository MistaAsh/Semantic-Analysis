AIM

  1. To understand the breakdown of Type Expressions to their simplified forms.
  2. To understand the structural differences between two type expressions and how their equivalence is checked.
  3. To observe and learn the implementation of Semantic Analysis phase using Type Checking.
  4. To visualize and structurally present how a compiler handles Type Conversion.
